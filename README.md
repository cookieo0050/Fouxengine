# Fouxengine

This engine is a fork of the Quake 3 Area engine, this engine lacks an enditor.
this engine supports Trenchbroom and NetRadiant

https://netradiant.gitlab.io/

https://trenchbroom.github.io/

### About

this engine tryes to improve performance and higher quality and tryes to make it a more modern engine.

this engine still is under the GLM licence.

for scripting it uses C/CPP just like Quake 3

#### Plane:
* Improve performance
* Improve Textures
* Add prost procecing graphics
* Reflections (Maybe)
